# Security Policy

We take the security of MediaForge seriously. Thank you for helping us keep this project secure.

---

## Supported Versions

Only the latest release version of MediaForge is currently supported with security updates. We encourage all users to run the latest code.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

---

## Reporting a Vulnerability

**Please do not open public GitHub issues for security vulnerabilities.**

If you discover a security vulnerability in this project, please report it confidentially:

1. Send an email to **keronpatel5656@gmail.com**.
2. Include a detailed description of the vulnerability, steps to reproduce, and a proof-of-concept if available.

We will acknowledge your report within 48 hours and work with you to analyze and patch the vulnerability. Once a fix is verified and pushed to the repository, we will publish a new release and credit you for the discovery.
